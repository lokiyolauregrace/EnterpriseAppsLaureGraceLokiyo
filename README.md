# Enterprise Apps Laure Grace Lokiyo

## Beschrijving

Prototype website voor Anderlecht Community NGO.

## Technologieën

- Java 17
- Spring Boot
- Thymeleaf
- Gradle

## Pagina's

- Home
- About
- Contact
- Nieuw evenement
- Details evenement

## Uitvoeren

1. Clone repository
2. Open project in IntelliJ
3. Run EnterpriseAppsLaureGraceLokiyoApplication
4. Open localhost:8081
